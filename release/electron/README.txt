========================================
  Todo List v1.1.0 - Electron 版本
  Windows x64 免安装便携版
========================================

【系统要求】
- Windows 10 或更高版本（64位）
- 无需安装额外运行时

【使用方法】
1. 解压到任意目录
2. 双击 TodoList.exe 启动应用
3. 首次启动会自动创建数据文件

【功能简介】
- 待办事项管理（创建、编辑、删除、状态切换）
- 标签系统（自定义标签、按标签筛选）
- Widget 桌面浮层（快速添加、查看即将到期事项）
- 深色/浅色主题切换
- 系统托盘（最小化后可从托盘恢复）
- 截止日期提醒（系统通知）
- 视图切换（卡片视图 / 紧凑视图）
- 已完成事项自动沉底 + 可折叠
- 快捷键：Ctrl+N 新建、Ctrl+F 搜索、Esc 关闭

【v1.1.0 更新内容】
- 修复：创建/编辑 Todo 弹窗无法操作的严重 Bug
- 新增：卡片/紧凑双视图切换
- 新增：已完成事项自动沉底 + 可折叠分隔头
- 新增：Widget 浮层已完成事项分组显示

【数据位置】
用户数据存储在：%APPDATA%\todolist\todos.json
建议定期备份此文件。

【已知问题】
- 首次启动可能需要几秒钟加载
