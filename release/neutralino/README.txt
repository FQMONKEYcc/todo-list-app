========================================
  Todo List v1.0.0 - Neutralinojs 版本
  Windows x64 免安装便携版
========================================

【系统要求】
- Windows 10 或更高版本（64位）
- 需要 WebView2 运行时（Win10/11 默认已安装）
  如果缺少 WebView2，请访问：
  https://developer.microsoft.com/zh-cn/microsoft-edge/webview2/

【使用方法】
1. 解压到任意目录
2. 双击 TodoList.exe 启动应用
3. 首次启动会自动创建数据文件

【文件说明】
- TodoList.exe     主程序（约 2.5 MB）
- resources.neu    应用资源包（约 400 KB）
- README.txt       本说明文件

注意：请确保 TodoList.exe 和 resources.neu 在同一目录下。

【功能简介】
- 待办事项管理（创建、编辑、删除、状态切换）
- 标签系统（自定义标签、按标签筛选）
- Widget 桌面浮层（快速添加、查看即将到期事项）
- 深色/浅色主题切换
- 截止日期提醒
- 快捷键：Ctrl+N 新建、Ctrl+F 搜索、Esc 关闭

【已知限制】
- Widget 浮层会在任务栏显示独立图标（框架限制）
- 系统通知能力受限（依赖操作系统 API）
- 系统托盘功能不如 Electron 版本完整

【数据位置】
用户数据存储在应用目录下的 .storage/ 文件夹。
建议定期备份此文件夹。

【版本信息】
- 版本号：v1.0.0
- 发布日期：2026-03-02
- 技术栈：Neutralinojs 6.5 + 系统 WebView2
- 总体积：约 3 MB

【优势】
相比 Electron 版本（~110 MB），Neutralinojs 版本仅约 3 MB，
启动更快（<1秒），内存占用更低（~50 MB）。
